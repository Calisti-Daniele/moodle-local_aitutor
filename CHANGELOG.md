## [0.1.0] — 2026-04-28

### Added / Aggiunto

- 🤖 Floating AI widget available on every Moodle page  
  Widget AI floating disponibile su ogni pagina Moodle
- 🧠 Context-aware AI: knows enrolled courses, grades, deadlines and certificates  
  AI contestuale: conosce corsi iscritti, voti, scadenze e certificati
- 🦙 Multi-provider support: Ollama (free, self-hosted), OpenAI, Anthropic  
  Supporto multi-provider: Ollama (gratuito, self-hosted), OpenAI, Anthropic
- 🔍 Automatic diagnostics page with one-click fixes  
  Pagina diagnostica automatica con correzioni in un click
- 🔒 GDPR compliant: full data export and deletion  
  Conformità GDPR: export e cancellazione dati completi
- 🌍 English and Italian translations  
  Traduzioni in inglese e italiano
- ⚙️ Admin settings page with live connection test  
  Pagina impostazioni admin con test connessione live
- 🔄 Auto-registration of web services on install/upgrade  
  Auto-registrazione dei web service all'installazione/aggiornamento
- 💬 Suggested questions on first open  
  Domande suggerite alla prima apertura
- 🌙 Dark mode support  
  Supporto dark mode
- 📱 Mobile responsive (fullscreen on small screens)  
  Responsive mobile (fullscreen su schermi piccoli)
